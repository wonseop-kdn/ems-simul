Q 01

Slack 버스와 위상각 기준점Slack Bus & Phase Angle Reference

왜 Slack 버스가 필요한가?

전력계통에서 뉴턴-랍슨 조류해석을 하려면 모든 모선의 전압 크기(V)와 위상각(θ)을 구해야 합니다. 그런데 전력 방정식은 위상각의 차이만 관계하기 때문에, 기준이 되는 절대 위상을 하나 고정해야 합니다. 이 기준 모선을 Slack(기준) 버스라고 합니다.

Slack 버스의 두 가지 역할
① 위상각 기준점: θslack = 0° 고정 (다른 버스는 이 기준과의 차이로 표현)
② 전력 균형자: 계통 내 미지의 총 발전량(P)과 무효전력(Q)를 자동으로 공급·흡수

Slack 버스는 어떻게 결정하나?

실무에서 Slack 버스는 보통 가장 큰 발전기(용량이 크고 예비력이 풍부한 기계)가 연결된 모선을 선택합니다. 이유는:

  • 부하·발전 불일치(오차)를 무한히 보상할 수 있을 만큼 용량이 커야 함
  • 수렴성: Slack 버스에서 계통 전체로 전류가 퍼져나가는 형태가 수치적으로 안정적

이 시뮬레이터에서는 Bus 1이 Slack입니다. Bus1은 전압 크기 V = 1.05 pu, 위상각 θ = 0°으로 고정되며, 필요한 유효·무효전력을 자동 공급합니다.

버스 유형알려진 값 (Known)구해야 할 값 (Unknown)
Slack Bus1V = 1.05 pu, θ = 0°P, Q (자동 결정)
PV Bus2P = 50 MW, V = 1.05 puθ, Q (자동 결정)
PQ Bus3,4,5P = −30/−25/−35 MW, Q = 계산V, θ
PQ Bus6,7P = 0, Q = 0V, θ

부하 버스의 P는 소비(음수 주입)으로 처리. PF = 0.95 가정, Q = P·tan(acos(0.95))

Q 02

위상각의 방향과 의미Phase Angle Direction & Physical Meaning

위상각이 음수라는 것은?

전류는 항상 위상이 앞선 곳(θ 높은 곳) → 위상이 뒤진 곳(θ 낮은 곳)으로 흐릅니다. Slack 버스(Bus1)를 θ = 0°으로 고정하면:

Bus1 (θ = 0.000°) → Slack, 전력 공급원 Bus6 (θ = −0.8°) → Bus1에서 받음 Bus7 (θ = −1.5°) → Bus6에서 받음 Bus3 (θ = −3.1°) → Bus7에서 받음 ← 가장 낮음 (부하 끝단) Bus5 (θ = −3.0°) → Bus7에서 받음 Bus4 (θ = −2.8°) → Bus7에서 받음 Bus2 (θ = +0.3°) → PV 발전기: 기준보다 앞섬 → 계통으로 전력 주입 (실제 시뮬레이션 결과, 부하 30/25/35 MW 기준)
직관적 이해: 위상각은 "전력이 내리막을 흐르는 경사"입니다.
발전기(Bus2: +0.3°) → 기준(Bus1: 0°) → 집전(Bus6: −0.8°) → 배전(Bus7: −1.5°) → 부하(Bus3∼5: −2.8 ∼ −3.1°)
경사가 급할수록 그 선로에 더 많은 유효전력이 흐릅니다.

위상각 차이와 유효전력의 관계

Pij ≈ (Vi · Vj / Xij) · sin(θi − θj) (저항 무시, 순수 리액턴스 선로 근사)

각도 차가 클수록(발전단 위상이 부하단보다 많이 앞설수록) 더 많은 유효전력이 전송됩니다. 대형 계통에서 30° 이상 벌어지면 안정도가 위험해집니다.

Q 03

퍼유닛(per-unit) 시스템Per-Unit System

왜 pu를 쓰는가?

실제 전력계통에는 765kV 송전선, 22.9kV 배전선, 0.22kV 가정용 선이 변압기로 연결되어 있습니다. 전압 레벨이 다른 구간을 실제 단위(kV, A)로 계산하면 변압기 권선비가 곱해져 식이 복잡해집니다. 퍼유닛(pu)은 각 구간의 정격값으로 나눠 무차원화하여 변압기를 "투명하게" 만듭니다.

전압 pu

구간마다 기준전압이 다릅니다.

V[pu] = V[kV] / V_base[kV] // 765kV 계통: V_base = 765kV // 배전: V_base = 22.9kV // 1.05 pu → 765×1.05 = 803.25 kV

전력 pu

전체 계통에서 기준 MVA는 하나.

P[pu] = P[MW] / S_base[MVA] // S_base = 100 MVA (고정) // 90 MW 부하 → 0.90 pu // 변압기가 있어도 동일 기준 사용
⚠ 자주 헷갈리는 부분: 전압 pu의 기준(V_base)은 변압기를 지날 때마다 변하지만, 전력 pu의 기준(S_base = 100 MVA)는 계통 전체에서 하나로 고정됩니다.

임피던스 pu 환산

Z[pu] = Z[Ω] × S_base / V_base² 예: X = 0.08 pu → 실제 리액턴스 = 0.08 × 765² / 100 = 468.7 Ω (765kV 계통)

이 시뮬레이터의 선로 임피던스는 이미 pu 값으로 입력되어 있어, 별도 환산 없이 Ybus 행렬을 구성할 수 있습니다.

Q 04

Ybus 어드미턴스 행렬Bus Admittance Matrix

Ybus란?

전력계통의 전기적 연결을 행렬로 표현한 것입니다. 옴의 법칙 I = Y·V에서 Y에 해당합니다. 복소 어드미턴스 y = g + jb = 1/(r+jx)를 사용합니다.

구성 규칙

  • 대각 요소 Yii = 버스 i에 연결된 모든 지선 어드미턴스의 합 (+)
  • 비대각 요소 Yij = 버스 i–j 사이 선로 어드미턴스의 부호 반전 (−)
  • 대칭 행렬: Yij = Yji

Q 05

Newton-Raphson 반복 수렴 추적Newton-Raphson Iteration Trace

NR의 기본 원리

전력 균형 방정식은 비선형입니다: Pcalc(V,θ) − Pspec = 0. 뉴턴-랍슨은 이를 선형화하여 반복 풀이합니다.

[ΔP] [H N] [Δθ ] [ΔQ] = [M L] [ΔV/V] → Jacobian · 수정량 = 불일치량 다음 반복: θ += Δθ, V += V·(ΔV/V) H = ∂P/∂θ, N = ∂P/∂(V/V), M = ∂Q/∂θ, L = ∂Q/∂(V/V)

실제 반복 수렴 (부하: 30/25/35 MW, PV: 50 MW)

초기값(플랫 스타트): V = 1.0 pu, θ = 0° for all buses

Iter 0
‖ΔP,ΔQ‖ = 0.900 초기 불일치 (부하 = 0.9 pu)
Iter 1
‖ΔP,ΔQ‖ = 0.0021 1회만에 99.8% 감소 (이차 수렴)
Iter 2
‖ΔP,ΔQ‖ = 3.1e-7 사실상 수렴
Iter 3
‖ΔP,ΔQ‖ = 6.8e-15 수치 오차 수준 (수렴 완료)
이차(quadratic) 수렴: 반복마다 유효자릿수가 두 배씩 늘어납니다. 0.9 → 0.002 → 3e-7 → 6e-15. 그래서 NR은 보통 3~5회 안에 수렴합니다.

Q 06

PV 버스와 AVR 자동전압조정기PV Bus & Automatic Voltage Regulator

PV 버스란?

PV 버스는 유효전력 P(발전량)와 전압 크기 V가 고정된 발전기 버스입니다. 무효전력 Q와 위상각 θ는 조류해석이 자동으로 결정합니다.

알고 있는 것 (Specified)

  • 유효 발전량: P = 50 MW
  • 단자 전압: V = 1.05 pu

조류해석이 결정하는 것

  • 위상각: θ = +0.31°
  • 무효 출력: Q = +19.2 Mvar

AVR이 Q를 어떻게 조절하나?

발전기 단자 전압이 설정값(1.05 pu)보다 내려가면 AVR은 계자전류(励磁電流)를 올립니다. 이로써 발전기가 무효전력(Q)을 더 많이 공급하여 전압을 회복시킵니다.

V_terminal < V_set → I_field ↑ → E_q ↑ → Q 공급 ↑ → V ↑ V_terminal > V_set → I_field ↓ → E_q ↓ → Q 흡수 ↑ → V ↓ E_q: 발전기 내부 기전력 (계자전류에 비례)
핵심 비유: AVR은 자동 수도꼭지입니다. 전압(수압)이 낮아지면 수도꼭지(계자전류)를 열어 무효전력(물)을 더 보내 압력을 회복합니다. PV 버스에서 V가 고정되어 있다는 것은 "이 수도꼭지가 항상 작동 중"임을 가정합니다.

Q 한계 초과 시

실제 발전기는 Q를 무한히 공급할 수 없습니다 (권선 가열 한계). Qmax를 초과하면 PV → PQ 버스로 전환되어 전압이 떨어집니다. 이 시뮬레이터는 한계 없이 PV로 유지하는 이상적 모델을 사용합니다.

Q 07

변압기·선로 손실 분석Transformer & Line Loss Analysis

손실 계산 방법

선로 손실은 양쪽에서 측정한 전력 흐름의 차이입니다:

P_loss = P_from + P_to (양방향 합, 방향 부호 포함) = |Pij| − |Pji| 실수부 손실: P_loss = |I|² · R = g_ij · (V_i² + V_j² − 2V_iV_j·cos(θ_i−θ_j))

합계—2.17 MW총부하의 2.4%
왜 TR(Line3)이 손실 1위인가?
TR은 모든 부하전력(90 MW)이 집중 통과하는 병목입니다. 전류는 조류의 제곱에 비례하므로, 통과 전류가 크면 P_loss = I²R이 급격히 커집니다. R이 작아도(0.01 pu) 전류가 크면 손실이 더 클 수 있습니다.

손실을 줄이려면?

  • TR 병렬화: 부하를 두 TR로 분담 → 각 TR 전류 반감 → 손실 1/2
  • 역률 개선(Q 보상): 부하 옆에 콘덴서 설치 → 선로 전류 감소
  • 전압 상승: V ↑ → 같은 P에서 I = P/V ↓ → I²R ↓
Q 08

N-1 원칙이란?

임의의 설비(선로, 변압기, 발전기) 하나가 고장나더라도 나머지 계통이 정상 운전을 유지해야 한다는 신뢰도 기준입니다. 북미 NERC, 한국 전력계통 신뢰도 기준 모두 N-1을 요구합니다.

Q 09

WLS 상태추정 원리Weighted Least Squares State Estimation

왜 상태추정이 필요한가?

SCADA 계측값에는 노이즈가 섞여 있고, 계측기가 고장나면 불량 데이터(Bad Data)가 들어옵니다. 상태추정은 여러 개의 계측값을 종합하여 가장 그럴듯한 계통 상태(전압, 위상각)를 추정합니다.

계측 vs 상태

계측량 z (약 30개)

  • 버스 전압 크기: V₁∼V₇
  • 버스 유효주입: P₁∼P₇
  • 버스 무효주입: Q₁∼Q₇
  • 선로 유효조류: P₁₂∼P₆₇ (양방향)
  • 선로 무효조류: Q₁₂∼Q₆₇

상태변수 x (13개)

  • 위상각: θ₂∼θ₇ (6개, θ₁=0 고정)
  • 전압: V₁∼V₇ (7개)
  • 합계: 13개 미지수

여유도(redundancy) = 30 − 13 = +17 → 과결정계

WLS 목적함수

minimize J(x) = Σᵢ wᵢ · [zᵢ − hᵢ(x)]² zᵢ: 계측값 hᵢ(x): 상태 x에서 계산한 이론값 wᵢ = 1/σᵢ² (계측 정확도의 역분산, 정밀할수록 가중치 큼) 최적 조건: ∂J/∂x = 0 → Hᵀ W H · Δx = Hᵀ W · r H = ∂h/∂x (Jacobian), r = z − h(x̂) (잔차)

나쁜 데이터 검출: χ² 검정

H₀: 불량 데이터 없음 → J(x̂) ~ χ²(dof) where dof = m − n 정상: J(x̂) < χ²₀.₉₉(dof) → "측정값 신뢰 가능" 불량: J(x̂) > χ²₀.₉₉(dof) → "Bad Data 있음!"
시나리오J(x̂)χ² 임계값판정
정상 (노이즈만) ≈ 25 34.8 ✅ 정상
불량 데이터 주입 (+45 MW) ≈ 1878 34.8 ⚠ 불량 감지

불량 계측 식별: LNR (최대 정규화 잔차)

rₙᵢ = |zᵢ − hᵢ(x̂)| / √Ωᵢᵢ Ω = S W S (잔차 공분산) 가장 큰 rₙ → 가장 의심스러운 계측기 (제거 후 재추정) 실험: P₇₋₃ 조류 계측에 +45 MW 주입 → 해당 계측기의 rₙ가 최대 → 올바르게 식별됨

무효전력과 전압의 관계Reactive Power & Voltage Control

유효전력 vs 무효전력

유효전력 P (실일, Active)

  • 실제 일을 함 (모터 회전, 열 발생)
  • 단위: MW
  • 위상각 차이로 전송
  • 장거리 전송 가능

무효전력 Q (허일, Reactive)

  • 일을 안 함, 에너지 왕복 (인덕터↔커패시터)
  • 단위: Mvar
  • 전압 크기를 결정
  • 장거리 전송 어려움 (손실 큼)

왜 Q가 전압의 주역인가?

전압강하 근사 (R≪X인 고압 계통): ΔV ≈ (P·R + Q·X) / V X >> R 이므로: ΔV ≈ Q·X / V ← Q가 지배! P를 2배 늘려도 ΔV 변화 작음 Q를 2배 늘리면 ΔV도 2배 → 전압 급락
비유: 물 배관 시스템
• 유효전력(P) = 흐르는 물의 양(유량)
• 무효전력(Q) = 배관 속 수압(압력)
• 수압(Q)이 낮으면 아무리 많은 물(P)도 끝단에 도달하기 전에 압력이 떨어짐
→ Q 공급 부족 = 전압 붕괴(Voltage Collapse) 위험

Q 부족 시 해결 방법

  • 병렬 콘덴서(Shunt Capacitor): 부하 근처에 설치, 지역적 Q 공급 → 전압 상승, 선로 전류 감소
  • SVC/STATCOM: 전력전자 기반, Q를 실시간 가변 조정
  • 발전기 Q 증대: 계자전류 ↑ → PV 버스 Q 공급 ↑ → 전압 회복
  • 부하 역률 개선: 역률을 0.95 → 0.99로 개선 시 Q 소비 60% 감소
전압 붕괴(Voltage Collapse): Q 공급이 수요를 따라가지 못하면 전압이 연쇄적으로 하강합니다. 1996년 미국 서부, 2003년 이탈리아 대정전의 주요 원인이 Q 부족에 따른 전압 붕괴였습니다.
